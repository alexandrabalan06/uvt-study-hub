This project presents a website for a tutoring organization, it allows students to check the schedule and to register as a member or as a tutor.
Implemented requirements checklist: four linked pages, "Skip to main content" accessibility link, table and form include accessible markup, and there is a CSS external stylesheet.
Known limitations: The "Join Us" form does not work, everything is just visual, it cannot store the data received from the user.
